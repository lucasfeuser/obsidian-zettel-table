## Installation

### From Obsidian Community Plugins

1. Open Settings > Community plugins
2. Search for "Zettel Table"
3. Click Install, then Enable

### Manual Installation

1. Download `main.js`, `manifest.json`, and `styles.css` (if applicable) from the [latest release](https://github.com/lucasfeuser/obsidian-zettel-table/releases/latest)
2. Create a folder `zettel-table` in your vault's `.obsidian/plugins/` directory
3. Copy the downloaded files into that folder
4. Restart Obsidian and enable the plugin in Settings > Community plugins
